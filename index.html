Tentang Website

Kami membuat website flip flop kalkulator ini sebagai upaya untuk memberikan pemahaman hasil dari reset dan set. Dengan demikian, website ini diharapkan dapat menjadi sumber belajar yang bermanfaat.

Penjelasan RS Flipflop


RS Flip Flop mempunyai 2 input yaitu, S=Set dan R=Reset. Mempunyai 2 output yaitu Q dan Q'. Bertindak sebagai 1 bit memori dengan output Q sebagai nilai bit tersebut.

Set (1): Mengatur keluaran (Q) menjadi logika positif (1) saat dipicu.
Reset (0): Mengatur keluaran (Q) menjadi logika negatif (0) saat dipicu.
Tetap: Mempertahankan keluaran (Q) pada kondisi sebelumnya, tidak mengalami perubahan saat dipicu.
Taktentu: menghasilkan keluaran(Q) yang tidak menentu dilambangkan dengan 'x'.

Bantuan


Masukkan Nilai Reset: Anda dapat memasukkan nilai 1 atau 0 pada kolom Reset untuk mengatur keluaran (Q) menjadi logika positif (1) saat dipicu.
Masukkan Nilai Set: Anda dapat memasukkan nilai 1 atau 0 pada kolom Set untuk mengatur keluaran (Q) menjadi logika negatif (0) saat dipicu.
